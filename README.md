# Página Personal – Armando Ángulo

Este repositorio contiene el código  de la **página personal de Armando Ángulo**.  
Aquí se muestra información académica, intereses de investigación, publicaciones, docencia y datos de contacto.

## Estructura del repositorio

```
├── index.html        # Página principal
├── style.css         # Estilos del sitio
├── img/              # Imágenes (fotos, logotipos, etc.)
└── README.md         # Documentación del repositorio
```

La página estará disponible en:

```
https://maufloresm.github.io/ArmandoAngulo/
```

## Cómo editar el sitio

Los contenidos se actualizan modificando los siguientes archivos:

- **index.html**: Información del profesor, secciones, textos y enlaces.  
- **style.css**: Estilos visualales (colores, tamaños, espaciados, fuentes).  
- **img/**: Carpeta donde deben colocarse las imágenes del sitio.  

Haz commit y push para actualizar el sitio automáticamente.
